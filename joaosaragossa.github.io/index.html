  • Recanto Domus

    O objetivo desse projeto foi remodelar o antigo site afim de atingir e conquistar novos clientes, criação de nova identidade visual, ampliar a capacidade de vendas e interação cliente/serviço, contemplando dispositivos móveis e clientes de língua espanhola e inglesa. Para isso utilizei os recursos do HTML 5 e CSS 3 e da linguagem de programação JavaScript através da biblioteca JQuery para criar uma experiência fluida e intuitiva baseada na abordagem de UX One Page e Mobile First. Gerenciamento de conteúdo utilizando CMS WordPress utilizando banco de dados MySQL.
